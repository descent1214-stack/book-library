# Мини-библиотека книг

Простое консольное приложение на Python для хранения списка книг.

## Возможности
- добавление книг
- просмотр списка
- поиск книг
- удаление книг
- отметка о прочтении

## Запуск программы

1. Установить зависимости:
pip install -r requirements.txt

2. Запустить программу:
python main.py
